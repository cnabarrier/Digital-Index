\## Always do this

1\. Inspect relevant files first. Never guess.

2\. Write a checklist plan to `tasks/todo.md`.

3\. Ask for approval before major changes.

4\. Implement smallest possible changes.

5\. Summarize each step and update docs.



\## Documentation requirement

Maintain `docs/architecture.md` describing how the app works end-to-end.
